# PBI Input — TASK-happiness-trap

## 依頼内容

書籍「幸福になりたいなら幸福になろうとしてはいけない」（ラス・ハリス著）のギャラリーエントリーとレビューページを追加する。

## 目的

- 新しい書籍レビューをサイトに公開する
- ギャラリーとレビューの相互リンクを構築する
- インフォグラフィック画像を含める

## 制約

- 既存UIコンポーネント・レイアウトは変更しない
- コンテンツデータ（markdown + 画像）のみ追加
- descriptionにタイトル・著者名を含めない
- レビュー本文はユーザー提供のテキストを使用

## Scope

### In scope
- 表紙画像の配置
- インフォグラフィック画像の配置
- ギャラリーエントリー作成
- レビューエントリー作成

### Out of scope
- UIコンポーネントの変更
- 既存エントリーの修正
- CSS/スタイルの変更
