# 読書withCoco Brand Strategy

この文書は、読書withCocoのブランド原典である。レビュー文、投稿文、画像プロンプト、サイト改修、導線設計で迷ったときは、先にこの文書を参照する。

## ブランド定義

読書withCocoは、単なる書評サイトではない。

難解な本や人文知を、忙しい大人の日常で使える「小さな武器」へ翻訳する場所である。目指す姿は「人生の修復工房」。読者が本を通じて視界を整え、今日より少し楽に生きるための「次の一歩」を持ち帰れることを最優先する。

## 中心思想

本は人生を一発で変える魔法ではない。

本は、日々の見方を少しずつ変える相棒である。読書withCocoは、知識の量や正しさだけで読者を圧倒しない。本から取り出すべきものは、読者の生活に残る問い、見方の変化、今日できる小さな行動である。

## 読者像

主な読者は、仕事、育児、家庭、体調、役割変化の中で、自分の軸を見失いかけている30代から40歳手前の大人である。

読書への関心はあるが、難解さ、時間不足、疲労感によって一歩を踏み出しにくい。知識を浴びたいのではなく、自分の生活に持ち帰れる言葉を探している。読書withCocoは、この読者を急かさず、見下さず、置き去りにしない。

## 提供価値

- 難しい知識を日常の言葉へ翻訳する
- 読者のモヤモヤを言語化する
- 本から「問い」「変化」「実生活への一歩」を取り出す
- 読者を煽らず、安心して前へ進める
- ココちゃんを通じて、知の入口をやわらかくする
- アフィリエイト誘導よりも、投稿単体で持ち帰れる価値を優先する

## 7つの知性

すべてのレビュー、投稿、サイト改修は、以下の7つの観点を意識する。

1. 思想: その本やコンテンツは、何を信じているか
2. ビジョン: 読後にどんな景色を見せるか
3. 志: 誰の、どんな痛みを和らげるか
4. 戦略: 読者をどの導線で理解や行動へつなげるか
5. 戦術: 今日できる具体行動に落ちているか
6. 技術: AI、Web、編集、情報設計を適切に使えているか
7. 人間力: 煽らず、見下さず、読者に寄り添えているか

## 4つの投稿軸

1冊の本から、以下の4軸で発信を作る。

1. 問い: その本が読者に何を問うているか
2. 変化: 読む前と後で世界の見え方がどう変わるか
3. 実生活: 今日の生活でどう使えるか
4. ココちゃん: 難しい内容をやわらかく受け止める入口

投稿やレビューが要約だけで終わりそうなときは、この4軸のどれが弱いかを見直す。

## ココちゃんの役割

ココちゃんは装飾ではない。難解な知をやわらかく翻訳する案内役である。

ココちゃんは、読者と本の間に立ち、難しい概念への緊張をほどく。専門知を幼くするのではなく、読者が自分の生活に引き寄せて考えられる入口を作る。画像やコピーでは、ココちゃんを「かわいい添え物」として置くのではなく、本から生まれた景色を読者に案内する存在として扱う。

## SNS と Web サイトの役割分担

- SNS: 読者のモヤモヤに短く触れ、問いや気づきの入口を作る場所。1投稿だけで価値が完結することを優先する。
- Web サイト: 本の背景、レビュー、ギャラリー、読後の余韻を深める場所。SNSで生まれた関心を、落ち着いた読書体験へつなげる。
- 共通方針: サイト誘導やアフィリエイトを主役にしない。まず投稿単体で、読者に問いか行動を渡す。

## 禁止事項

- 「読まないと損」と煽る
- 専門用語で権威性を見せつける
- アフィリエイト導線を主役にする
- 読者を置き去りにした抽象論で終わる
- 本の要約だけで終わる
- 自分の感想だけで閉じる
- ココちゃんを単なる飾りとして扱う
- generic SaaS 風、量産型LP風、機能一覧風にブランドを寄せる
